# Sorting-numbers

## Опис
У цій програмі я створюю масив із 10 випадкових чисел від -50 до 50. Потім зробила функцію sorting, яка сортує цей масив простим методом порівняння: кожен елемент перевіряється з усіма наступними, і якщо число більше — вони міняються місцями. У результаті масив виходить відсортованим від меншого до більшого.
